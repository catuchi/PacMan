# Pacman

> Pacman game using Vanilla Javascript

## Demo

This app is deployed on netlify. Try it [Pacman](https://whimsical-cobbler-7797c5.netlify.app/)

## Run Locally

```bash
# Install dependencies
npm install

# Watch Parcel
npm run start
```

## Tech Stack

Javascript, HTML, CSS, Parcel Bundler
